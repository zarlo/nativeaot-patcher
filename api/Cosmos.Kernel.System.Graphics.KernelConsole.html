Class KernelConsole

Namespace
Cosmos.Kernel.System.Graphics
Assembly
Cosmos.Kernel.System.dll

Cell-based graphics console for kernel output. Uses a character grid (cells) similar to Aura OS for efficient terminal rendering. Can be instantiated on any canvas (hardware or virtual).
