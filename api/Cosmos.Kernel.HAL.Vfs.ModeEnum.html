Remarks

Layout mirrors Unix mode bits:

  • Permissions: owner/group/other read, write, execute (bits 0-8)
  • Special: sticky, setgid, setuid (bits 9-11)
  • File type: high nibble encodes the type (bits 12-15); mask with FileTypeMask.
